# Hosted downloads

Place **`VideoMaster.dmg`** in this folder so it is served at:

`/downloads/VideoMaster.dmg`

The VideoMaster marketing page links to that URL for **Download for macOS**.

After adding or replacing the DMG, deploy as usual. Do not commit large binaries if you prefer to upload the DMG via your host’s file UI instead; keep this path consistent so the site link stays valid.
